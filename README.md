

### NFSe Carioca

Requisitos:

 - PHP 7.4 
 - MySQL - Qualquer versão. 

 Instalação:

 - Baixe o projeto. 
 - Mova-o para a pasta `/xampp/htdocs`
 - Aponte o apache para a pasta raiz deste projeto `webroot`
 - Reinicie o servidor apache2 
 - Acesse o MySQL 
 - Crie a tabela `nota_fiscal` com o script `database.sql`
